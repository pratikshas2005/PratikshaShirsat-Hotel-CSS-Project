# Hotel Website - CSS Mini Project

Student Name: Pratiksha Shirsat

 Project Description
This is a mini CSS project created for the Web Development module.  
The project is a simple hotel website designed using **pure HTML and External CSS** (no frameworks, no JavaScript).  
The website contains 2–3 pages including:

- Home Page  
- Rooms Page  
- About Page (optional)

 CSS Concepts Implemented
- Box Model (margin, padding, border)
- Typography (font-family, size, weight, alignment)
- Background properties (color, images)
- Flexbox Layout
- Pseudo Classes (:hover, :first-child, :nth-child)
- Hover Effects (buttons, images, navbar links)
- Clean spacing & layout design
